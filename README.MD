## Abstract
다양한 공공 데이터를 활용해 서울시 공공자전거 '따릉이'의 이용 요인을 분석하고 '따릉이' 대여소의 입지에 분석된 결과를 활용해 본다

## Main Goal
* 서울시 공공자전거 '따릉이' 대여소의 최적입지 선정
* '최적입지'란 예상 총 이용률이 최대가 되는 복수입지
* 기존 대여소 위치 고려해 일부 재배치 제안
